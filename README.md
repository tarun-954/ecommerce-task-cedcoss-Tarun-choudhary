# ecommerce-task-cedcoss-Tarun-choudhary

Round 2 task project for Cedcoss x LPU placement drive.

## Tech Stack
- React.js (frontend)
- Node.js + Express (backend)
- MongoDB (database)

## Project Modules
1. Module 1: Base project structure ✅
2. Module 2: Backend API (reviews CRUD)
3. Module 3: Frontend UI for ratings + reviews
4. Module 4: Helpful votes (bonus)
5. Module 5: Final polish + process docs

## Run (after setup in next modules)
Backend:
```bash
cd server
npm install
npm run dev
```

Frontend:
```bash
cd client
npm install
npm run dev
```
